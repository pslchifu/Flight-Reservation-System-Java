# Flight Reservation System (Java)

Basic console-based flight reservation system developed in Java. 

## How to run
1. Open the project in your IDE (e.g., VS Code / IntelliJ/ NetBeans/ etc).
2. Run `Main.java`.
   
## Features
- View available flights
- Make and cancel reservations
- Track seat availability
- Interactive menu for users

## Technologies
- Java
- Object-Oriented Programming (OOP)

## What I learned
- Java fundamentals
- OOP concepts (classes, objects, methods)
- Program structure and logic
